## Установка

```bash
# Клонируем приложение
git clone https://github.com/xzdwq/netwrix-test.git
cd netwrix-test
# Установка зависимостей backend
npm i
# Установка зависимостей frontend
cd ui
npm i
cd ..
```

## Создание базы данных
В качестве базы данных использовалась MySQL/MariaDB-10.4
```
host: locolhost
port: 3306
```
Создать базу данных `netwrix`  
Кодировка: `utf8mb4_unicode_ci`  
В проекте настроена автосинхронизация сущностей (таблиц), т.е. создавать вручную таблицы не нужно (оставить БД пустой)  

## Запуск приложения
Из корня приложения (./netwrix-test)
```bash
npm run watch
```
Команда запустит backend и frontend.  
Web приложение будет доступно по адресу `localhost:3000`  

## Миграция данных в базу данных
После того как приложение запустилось (произошла автосинхронизация сущностей и создалить таблицы), в среде выполнения SQL на нашем сервере базы данных MySQL перейти в ранее созданную базу данных `netwrix` и выполнить SQL-запрос из файла `./netwrix-test/migration/test.sql`
